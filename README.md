# 🚀 DP-Fácil

**DP-Fácil** é uma plataforma completa de **Departamento Pessoal**, feita do zero para automatizar cálculos de **folha de pagamento**, **férias**, **13º salário**, **rescisões** e outras rotinas trabalhistas, tudo de forma **simples, rápida e visual**.

---

## 🎯 Objetivo do Projeto
O DP-Fácil foi desenvolvido para:
- Facilitar o dia a dia de profissionais de Departamento Pessoal.
- Servir como ferramenta educativa para estudantes e iniciantes.
- Automatizar cálculos trabalhistas de forma confiável e prática.

---

## ⚡ Funcionalidades
- Cálculo automático de folha de pagamento e pró-labore
- Controle de férias e 13º salário
- Simulação de rescisões
- Histórico de funcionários
- Exportação de relatórios em PDF e Excel
- Interface web responsiva, limpa e intuitiva
- **Fácil de usar**, mesmo para quem está aprendendo

---

## 🛠️ Tecnologias Utilizadas
- **Front-end:** HTML, CSS (Tailwind), JavaScript
- **Back-end:** Planejado para Python/Firebase (futuro)
- **Exportação de arquivos:** PDF/Excel
- **Controle de dados:** Local storage e planejado para banco em nuvem
